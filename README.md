# AMP Music Player
The Ambient Music Player is a music player that simultaneously plays audio and video.\
**Features:**
* Audio
    * Play MP3 files
    * Select a custom music directory
    * Shuffle collection
* Video
    * Play TS files
    * Shuffle collection

**Dependencies:**
* FFMpeg
* FFPlay

There are currently no releases, it is still a WIP existing only as a java program.
